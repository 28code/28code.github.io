4.7. 检查net.bridge设置

检查
net.bridge.bridge-nf-call-arptables = 1
net.bridge.bridge-nf-call-ip6tables = 1
net.bridge.bridge-nf-call-iptables = 1
是否为1

如果不是1
修改方式：修改 /etc/sysctl.conf 文件
或者新建 /etc/sysctl.d/k8s.conf 增加
net.bridge.bridge-nf-call-arptables = 1
net.bridge.bridge-nf-call-ip6tables = 1
net.bridge.bridge-nf-call-iptables = 1
让文件生效
sysctl -p /etc/sysctl.d/k8s.conf
